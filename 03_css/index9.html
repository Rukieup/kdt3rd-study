CSS display 속성에 대해 알아보자

  • 요소를 화면에 어떻게 보여줄지 결정하는 속성
  • html 태그는 block과 inline 요소 나눔

inline

  1. span, a, img
  2. content(내용물) 만큼 공간 할당 -> width, height, padding, margin 사용할 수 없음(width 등 사이즈 정의하는게 의미없음)
  3. 줄바꿈 없이 바로 앞에 배치
d-inline1 클래스를 갖는 div1
d-inline2

block

  1. ex: div, p, ul, ol
  2. 한 줄 을 다 차지함 -> width, height, padding, margin 사용이 자유
d-block1 클래스를 갖는 div
d-block1 클래스를 갖는 div

inline block

  1. block요소처럼 크기 지정을 하되
  2. inline요소처럼 다른 내용들과 같은 줄을 공유하고 싶은 경우
  3. block과 inline 요소의 장점을 가짐
d-inline-block1 클래스를 갖는 div
d-inline-block2

none

  1. 요소를 화면에 표시하지 않음
  2. 주의) display: none; 요소가 차지하는 공간도 사라짐
  3. 요소를 그리지 않음(랜더링하지 않음)
